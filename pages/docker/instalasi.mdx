# Instalasi Docker

Berikut adalah langkah-langkah instalasi Docker di berbagai sistem operasi.

## Windows

<Callout type='warning'> Pastikan sebelum melakukan instalasi sudah memiliki akun DockerHub terlebih dahulu<br/>Jika belum silahkan membuat akun disini: [https://hub.docker.com/](https://hub.docker.com/)</Callout>

1. Buka [halaman unduh Docker Desktop](https://docs.docker.com/desktop/install/windows-install/) menggunakan browser.
2. Download _installer_ Docker dengan nama **Docker Desktop for Windows - x86_64**.
3. Buka file yang telah diunduh.
4. Ikuti langkah-langkah instalasi yang ditampilkan oleh _installer_ Docker.
5. Setelah instalasi selesai, buka Docker Desktop dari _Start Menu_.

Referensi: [https://docs.docker.com/desktop/install/windows-install/](https://docs.docker.com/desktop/install/windows-install/)
